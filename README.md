# EDA - Carga de Energia ONS

Análise exploratória de dados utilizando dados reais de carga energética do ONS referentes ao ano de 2024.

## Tecnologias
- Python
- Pandas
- Matplotlib
- Seaborn

## Estrutura
- data/raw: datasets originais
- notebooks: notebooks da análise
- reports/figures: gráficos gerados
